# gmi: a Gemini Capsule Generator

**gmi** is a Gemini capsule generator to easily manage articles and microblog posts from the terminal.

This is work in progress.

## Prerequisites

**gmi**

### Dependencies

bash, coreutils, findutils, fzf, rsync, sed, ssh
